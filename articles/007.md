## 結論｜ベランダ雨漏りは原因を見誤らないことが重要です
ベランダ雨漏りで大切なのは、症状が出ている場所だけを補修しないことです。たとえば、室内のシミだけを見て屋根だけ直しても、実際には外壁のひび割れやサッシまわりのコーキング劣化が原因だったというケースがあります。

特に松山市では、普段の雨では症状が出ず、台風や横殴りの雨のときだけ雨漏りすることがあります。その場合、原因箇所を目視だけで判断しにくいため、屋根・外壁・サッシ・防水・付帯部を総合的に確認する必要があります。

早めに点検すれば、部分補修で済む可能性があります。反対に、長く放置すると下地材や断熱材、室内クロスまで傷み、補修費用が高くなることがあります。

## ベランダ雨漏りでよくある症状
次のような症状がある場合は、早めに確認しましょう。

- ベランダ下の天井にシミがある
- ベランダ床にひび割れや膨れがある
- 排水口まわりに水たまりが残る

これらの症状は、単なる汚れや一時的な湿気に見えることもあります。しかし、雨の日や台風後に繰り返し発生する場合は、雨水が建物内部へ入り込んでいる可能性があります。

## 放置するとどうなる？
ベランダ雨漏りを放置すると、表面のシミだけでなく、建物内部の下地や木部まで傷むことがあります。

放置リスクは以下です。

| 放置リスク | 内容 |
|---|---|
| 下地の腐食 | 木部や野地板が水を含み、補修範囲が広がる |
| カビの発生 | 室内の湿気や臭いにつながる |
| 断熱材の劣化 | 壁内や天井裏の断熱性能が下がる |
| シロアリリスク | 湿った木部が害虫被害につながる可能性 |
| 補修費用の増加 | 部分補修で済まず、大きな工事になることがある |

小さな症状でも、繰り返し発生している場合は早めの点検が安心です。

## 自分でできる応急処置
雨漏りが疑われる場合、自分でできるのは室内側の被害を広げない対応までです。

- 濡れている場所にバケツやタオルを置く
- 家具や家電を水から離す
- 電気まわりに水が近づかないようにする
- 症状が出た場所を写真に残す
- 雨の強さや風向きをメモする
- 無理に屋根や高所へ上がらない

屋根に上がる、外壁を自己判断で塞ぐ、雨の日に高所作業をするのは危険です。また、原因を確認しないままコーキングを打つと、後から調査しにくくなることがあります。

## 同時に確認したい部位

ベランダ雨漏りを点検するときは、原因箇所だけでなく周辺部位もあわせて確認すると安心です。雨水は建物内部を伝って移動するため、一見関係なさそうな場所が原因になっていることがあります。

| 同時に確認したい部位 | 確認する理由 |
|---|---|
| 屋根まわり | 割れ、ズレ、板金浮きが雨漏り原因になるため |
| 外壁まわり | ひび割れやシーリング劣化から水が入るため |
| サッシまわり | 横殴り雨で浸水しやすい箇所のため |
| 雨樋・排水まわり | 詰まりや外れで水が回り込むことがあるため |
| 軒天・破風 | 雨漏りや湿気のサインが出やすいため |

部分的な補修で済むか、周辺まで工事した方がよいかは、建物の状態によって変わります。写真付きで説明を受けながら判断しましょう。
