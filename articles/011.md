## 屋根修理が必要になる原因

屋根修理の原因は一つとは限りません。複数の劣化が重なって、雨水の侵入につながることがあります。

| 原因 | 起こりやすい症状 | 確認ポイント |
|---|---|---|
| 屋根材の割れ・ズレ | 雨漏り、屋根の見た目の乱れ | 瓦、スレート |
| 棟板金の浮き | 強風時の音、飛散リスク | 屋根の頂点部分 |
| 谷板金の劣化 | 雨量が多い日の雨漏り | 雨水が集まる谷部分 |
| 防水紙の劣化 | 再発する雨漏り | 屋根下地 |
| 雨樋の不具合 | 外壁への雨水の回り込み | 軒先、排水経路 |

| 板金補修 | 棟板金や谷板金を補修 | 3万円〜30万円程度 |
| 屋根カバー工法 | 既存屋根の上に新しい屋根材を施工 | 80万円〜 |
| 葺き替え | 屋根材と下地を交換 | 100万円〜 |

費用は建物の状態、足場の有無、下地の傷み具合、使用する材料によって変わります。正確な金額は、現地調査で原因箇所と補修範囲を確認してから判断しましょう。
