## 結論｜漆喰補修は雨漏り前の点検が重要です
漆喰補修は、雨漏りが起きてからではなく、劣化のサインが出た段階で確認することが大切です。屋根の不具合は室内から見えにくいため、気づいたときには下地まで水が回っていることがあります。

特に漆喰・棟瓦・内部土・防水紙は、屋根の防水性に関わる重要な部分です。小さなズレや浮きでも、台風や強風の際に雨水が入り込み、雨漏りの原因になることがあります。

早めに点検すれば、部分補修で済む可能性があります。放置して劣化が進むと、広範囲の屋根工事や足場工事が必要になる場合があります。

## 漆喰補修が必要になる主な症状
次のような症状がある場合は、屋根点検を検討しましょう。

- 屋根の漆喰が剥がれている
- 棟まわりの白い部分が崩れている
- 瓦屋根の雨漏りが心配

屋根の症状は地上から見えにくいことが多いため、双眼鏡や写真で確認できる範囲にとどめ、無理に屋根へ上がらないようにしてください。

## 松山市で屋根劣化が進みやすい理由
松山市では、季節ごとの気候により屋根に負担がかかります。夏場の紫外線は屋根材や塗膜を劣化させ、台風時の強風は板金の浮きや瓦のズレを引き起こすことがあります。

また、三津・堀江・北条など海に近い地域では、金属部材の錆やビスの劣化にも注意が必要です。住宅密集地や風の通り道では、横殴り雨で屋根の取り合い部分から水が入りやすくなることもあります。

屋根は建物を雨から守る最前線です。小さな劣化でも、早めに確認することで大きな雨漏りを防ぎやすくなります。

原因は一箇所とは限りません。瓦、板金、漆喰、防水紙、下地などが複合的に劣化している場合もあります。

## 放置するとどうなる？
漆喰補修を放置すると、雨漏りや屋根材の飛散につながる可能性があります。

| 放置リスク | 内容 |
|---|---|
| 雨漏り | 屋根の隙間から雨水が入り、天井シミにつながる |
| 下地腐食 | 野地板や木部が水を含み、補修範囲が広がる |
| 屋根材の飛散 | 台風や強風で部材が飛ぶ危険がある |
| 室内被害 | 天井クロスや断熱材の補修が必要になる |
| 費用増加 | 部分補修で済まず、大きな工事になることがある |

台風後に屋根の異音やズレが気になった場合は、早めの点検が安心です。

## 屋根点検で確認するポイント
屋根点検では、次のような箇所を確認します。

- 屋根材の割れ、ズレ、浮き
- 板金の浮き、錆、釘抜け
- 漆喰の剥がれ、崩れ
- 谷部分の水はけ
- 雨樋の詰まりや外れ
- 軒天のシミや黒ずみ
- 小屋裏の雨染み

屋根は部位ごとに役割が違うため、気になる箇所だけでなく全体を確認することが大切です。

## 業者に相談するときの注意点
漆喰補修を依頼するときは、見積もり金額だけで判断しないようにしましょう。重要なのは、どこが劣化していて、どの範囲を補修するのかが明確になっていることです。

確認したい項目は以下です。

- 劣化箇所を写真で説明してくれるか
- 応急処置か根本修理かを分けて説明してくれるか
- 足場の必要性が明確か
- 使用する材料や工法が分かるか
- 追加費用が出る可能性を説明してくれるか
- 工事後の保証やアフター対応があるか

## 点検前に確認しておきたいこと

業者へ相談する前に、以下を整理しておくと現地確認がスムーズです。

- 漆喰の剥がれ・崩れ

写真がある場合は、雨の直後や症状が分かりやすいタイミングで残しておきましょう。調査日に乾いていても、写真があれば雨水の出方や範囲を確認しやすくなります。

## 早めに相談した方がよいサイン

次のような変化がある場合は、できるだけ早めに点検を受けることをおすすめします。

| サイン | 確認したいこと |
|---|---|
| 雨のたびに症状が出る | 一時的な濡れではなく、雨水の通り道ができている可能性 |
| 台風後から急に症状が出た | 強風で部材が浮いたり、隙間が広がった可能性 |
| シミや黒ずみが広がっている | 内部で水が回っている可能性 |
| カビ臭さや湿気が残る | 壁内や天井裏に湿気がこもっている可能性 |
| 一度補修したのに再発した | 原因が別の場所に残っている可能性 |

症状が小さいうちに確認できれば、部分補修で済む場合があります。反対に、放置して下地まで傷むと、工事範囲が広がりやすくなります。

## 費用を抑えるための考え方

費用を抑えるには、単に安い見積もりを選ぶのではなく、原因を正しく確認して再発を防ぐことが大切です。安い補修でも原因が残っていれば、再調査や再工事が必要になり、結果的に高くなることがあります。

また、足場が必要な工事では、屋根・外壁・雨樋・軒天・シーリングなど、同時に点検できる箇所を確認しておくと効率的です。今すぐ直す箇所と、次回メンテナンスでよい箇所を分けることで、無駄な工事を避けやすくなります。

## 見積もりで確認したいポイント

見積もりでは、総額だけでなく、何をどこまで直すのかを確認することが大切です。

| 確認項目 | 見るポイント |
|---|---|
| 原因説明 | 写真付きで根拠を説明しているか |
| 工事範囲 | 部分補修か、広範囲の補修か |
| 使用材料 | シーリング材、防水材、板金材などが明記されているか |
| 足場 | 必要性と費用が分かるか |
| 追加費用 | 下地補修などの可能性を説明しているか |
| 保証・アフター | 工事後の対応範囲が分かるか |

特に雨漏りが関係する工事では、原因を見誤ると再発することがあります。見積もりの根拠を写真で確認し、必要な工事だけを選べるようにしましょう。

## 同時に確認したい部位

漆喰補修を点検するときは、原因箇所だけでなく周辺部位もあわせて確認すると安心です。雨水は建物内部を伝って移動するため、一見関係なさそうな場所が原因になっていることがあります。

| 同時に確認したい部位 | 確認する理由 |
|---|---|
| 屋根まわり | 割れ、ズレ、板金浮きが雨漏り原因になるため |
| 外壁まわり | ひび割れやシーリング劣化から水が入るため |
| サッシまわり | 横殴り雨で浸水しやすい箇所のため |
| 雨樋・排水まわり | 詰まりや外れで水が回り込むことがあるため |
| 軒天・破風 | 雨漏りや湿気のサインが出やすいため |

部分的な補修で済むか、周辺まで工事した方がよいかは、建物の状態によって変わります。写真付きで説明を受けながら判断しましょう。

## 依頼から点検・見積もりまでの流れ

初めて相談する場合は、どのように進むのか分からず不安に感じる方もいます。一般的な流れは次の通りです。

1. 気になる症状を問い合わせる
2. 現地で症状や外装の状態を確認する
3. 劣化箇所を写真で共有する
4. 必要な工事と様子見でよい箇所を分けて説明する
5. 見積もり内容を確認する
6. 納得したうえで工事を依頼する

相談したからといって、すぐに契約しなければならないわけではありません。まずは状態を知り、必要な対応を整理することが大切です。


## 屋根まわりの点検で大切な視点

屋根の不具合は、表面の部材だけで判断できないことがあります。屋根材の下には防水紙や下地材があり、雨水が入り込んでいる場合は、見える部分より内部の傷みが進んでいることもあります。

そのため、漆喰補修では、表面の補修だけでなく、周辺の板金、瓦、漆喰、雨樋、小屋裏の雨染みまで確認することが大切です。見積もり時には、どこまで確認したうえで工事内容を決めているのかを聞いておきましょう。

## よくある質問
### 屋根に上がって自分で確認してもよいですか？

危険なためおすすめしません。屋根は滑りやすく、転落事故につながる可能性があります。確認は専門業者へ依頼しましょう。

### 小さなズレや割れでも修理が必要ですか？

雨水の侵入口になる可能性があるため、早めの確認をおすすめします。症状が軽いうちなら部分補修で済む場合があります。

### 足場は必ず必要ですか？

工事内容や屋根の高さによって変わります。安全に作業するために足場が必要になることがあります。

### 雨漏りしていなくても点検できますか？

できます。雨漏り前に点検することで、劣化の早期発見や予防につながります。
