## 結論｜雨漏り調査は原因特定が命です

雨漏り調査で最も大切なのは、雨水の侵入口を正しく確認することです。

室内に出ている水は、建物内部を伝って移動していることがあります。たとえば、天井にシミがある場合でも、原因は屋根材の割れだけとは限りません。外壁のひび割れ、サッシまわりのコーキング切れ、ベランダ防水、笠木や幕板の隙間が関係していることもあります。

そのため、雨漏り調査では次のような考え方が必要です。

| 見る場所 | 確認する内容 |
|---|---|
| 室内 | シミ、濡れ、カビ、クロスの浮き |
| 小屋裏 | 雨染み、水の流れ、木部の変色 |
| 屋根 | 瓦のズレ、板金浮き、屋根材の割れ |
| 外壁 | ひび割れ、塗膜劣化、サイディング浮き |
| サッシ | コーキング劣化、取り合い部の隙間 |
| ベランダ | 防水層、排水口、立ち上がり |
| 付帯部 | 軒天、笠木、幕板、雨樋まわり |

## 松山市で雨漏り原因が分かりにくい理由

松山市の雨漏りは、普段の雨では出ず、台風や強風を伴う雨のときだけ発生することがあります。これは、横から雨が吹き込むことで、通常では入りにくい外壁目地やサッシまわりから雨水が入るためです。

また、松山市内でも地域によって住宅への影響は変わります。

- 三津・堀江・北条など海に近い地域では金属部材の錆に注意
- 久米・石井・来住町など住宅密集地では外壁やベランダ劣化に注意
- 山越・道後・祝谷など傾斜地や風の通り道では横殴り雨に注意
- 余戸・空港通周辺では風を受けやすい面の外壁目地に注意

こうした地域事情を踏まえて調査することで、原因の見落としを減らせます。

## 雨漏り調査の主な方法

### 目視調査

目視調査は、屋根・外壁・ベランダ・サッシまわりを見て、劣化や破損がないか確認する基本調査です。コーキングの切れ、外壁のひび割れ、屋根材のズレ、板金の浮き、防水層のひび割れなどを確認します。

### 散水調査

散水調査は、雨漏りが疑われる箇所に水をかけて、雨漏りが再現するか確認する方法です。実際の雨に近い状態をつくれるため、原因箇所を絞り込みやすくなります。

ただし、水をかける順番や時間を間違えると、原因が分かりにくくなります。経験のある業者に依頼することが大切です。

### 赤外線調査

赤外線カメラを使い、壁や天井の温度差から水分がある可能性を確認する方法です。内部の水分を推測しやすい一方で、赤外線だけで原因を断定できるわけではありません。目視や散水調査と組み合わせて判断します。

### 写真付き診断

調査後に、劣化箇所を写真で共有してもらうことも大切です。屋根や外壁は自分で確認しにくいため、写真があると「なぜこの修理が必要なのか」が分かりやすくなります。

## 調査前に準備しておくとよいこと

雨漏り調査を依頼する前に、以下を整理しておくとスムーズです。

- いつから症状が出ているか
- どの部屋、どの天井、どの窓で起きているか
- 雨のたびに出るのか、強い雨のときだけか
- 風向きによって症状が変わるか
- 台風や大雨の後に発生したか
- 過去に屋根・外壁・防水工事をしたか
- 症状が出たときの写真があるか

調査日に症状が乾いていても、写真があれば状況を把握しやすくなります。

## 調査後に必ず確認すべきこと

雨漏り調査後は、見積もり金額だけでなく、以下を確認しましょう。

| 確認項目 | 見るポイント |
|---|---|
| 原因箇所 | どこから雨水が入っている可能性があるか |
| 根拠 | 写真や調査結果で説明されているか |
| 修理方法 | 応急処置か根本修理か |
| 修理範囲 | どこまで直すのか |

| 保証 | 工事後の対応範囲はどうか |

「ここが原因だと思います」という説明だけでは不十分です。写真や調査内容をもとに、なぜその工事が必要なのかを確認することが大切です。

実際には、屋根から入った雨水が壁の中を伝って窓まわりに出ることもあります。外壁目地から入った水が、室内の天井付近に回ることもあります。ベランダの防水層が原因なのに、外壁のひび割れだけを補修して再発するケースもあります。

## 調査後の提案で確認したいこと

調査後に提案を受けるときは、「どの工事をするか」だけでなく、「なぜその工事が必要なのか」を確認しましょう。

たとえば、コーキング打ち替えを提案された場合は、どの目地が切れているのか、部分補修でよいのか、外壁全体のシーリングが劣化しているのかを確認します。屋根補修を提案された場合は、瓦や板金のどこが傷んでいるのか、足場が必要なのかを確認します。

必要な工事だけを選ぶためには、調査結果と見積もり内容がつながっていることが大切です。調査内容と関係のない大きな工事をすすめられた場合は、理由を聞いてから判断しましょう。
