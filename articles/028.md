## 笠木雨漏りが起きやすい理由

笠木は雨を受けるだけでなく、外壁内部への水の侵入を防ぐ役割があります。継ぎ目や端部の小さな隙間でも、雨水が壁内へ回ることがあります。

松山市では、普段の雨では症状が出ず、強い風を伴う雨のときだけ雨漏りするケースがあります。原因を見つけるには、症状が出た条件と外装の劣化状況を合わせて確認することが大切です。

## 笠木雨漏りでよくある症状

- ベランダ下にシミがある
- 笠木の継ぎ目に隙間がある
- 手すり下が黒ずむ
- 台風後に室内側が湿る

一度だけの症状に見えても、雨のたびに同じ場所が濡れる場合は、建物内部に雨水の通り道ができている可能性があります。

## 主な原因

| 原因 | 起こりやすい症状 | 対応の考え方 |
|---|---|---|
| 笠木継ぎ目の劣化 | 外壁内部への浸水 | シーリング補修 |
| ビスまわりの隙間 | 手すり下の雨染み | 防水処理 |
| 防水立ち上がり不足 | ベランダ下の雨漏り | 防水補修 |
| 外壁取り合い不良 | 壁内雨染み | 外壁補修 |
| 笠木板金の浮き | 横殴り雨で浸水 | 板金補修 |

原因は一つとは限りません。外壁、屋根、サッシ、防水、板金が複合的に関係している場合もあります。

## 修理方法と費用目安

| 修理方法 | 内容 | 費用目安 |
|---|---|---|
| シーリング補修 | 継ぎ目や端部を補修 | 3万円〜15万円程度 |
| 笠木板金補修 | 浮きや隙間を補修 | 5万円〜30万円程度 |
| ベランダ防水補修 | 立ち上がりまで補修 | 8万円〜30万円程度 |
| 外壁補修 | 雨水が回った箇所を補修 | 5万円〜30万円程度 |

費用は足場の有無、下地の傷み、補修範囲によって変わります。正確な金額は現地調査後に確認しましょう。

## 応急処置でやってよいこと・避けたいこと

室内に水が出ている場合は、まず被害を広げないことが大切です。濡れている場所にタオルやバケツを置き、家具や家電を離し、電気まわりに水が近づかないようにしましょう。

一方で、高所作業や自己判断のコーキングは避けてください。原因が分からないまま隙間を塞ぐと、別の場所へ水が回ったり、後から調査しにくくなったりすることがあります。

## 点検時に確認したい場所

| 部位 | 確認する理由 |
|---|---|
| 屋根まわり | 雨水が壁内へ回ることがある |
| 外壁目地 | シーリング劣化が雨漏り原因になる |
| サッシまわり | 横殴り雨で浸水しやすい |
| ベランダ防水 | 下階天井のシミにつながる |
| 雨樋・軒天 | 水の流れや湿気のサインが出やすい |
